
# What this is

This repository contains a list of globally banned Steam players.

Players are added here when they deliberately try to grief on Steam lobbies. This list is then periodically fetched by the Mindustry client on Steam.

This list also contains the IDs of me (Anuke) and @buthed010203 (the main developer of Foo's Client), which makes us admins in **all** Steam lobbies. This helps assist us in catching and banning these griefers in lobbies.

# Why this exists

On Steam, some people seem to enjoy going to every single public Steam lobby and deliberately sabotaging people's bases for fun. Since the rate limit doesn't apply to Steam servers (for good reason), this can be quite devastating. Thus, this list was created to flag such players.

# What this is not

- This **does not** affect any dedicated server you are hosting; it only works through Steam lobbies.
- This **is not** a backdoor, as it doesn't let the admins do anything important other than ban/kick people from lobbies (or run waves, if you care about that...?)
- This **does not** make us impervious to bans or kicks; if the host wants, they can kick or ban the admins contained in this list when they join.
- This list **does not** have random people set as admins, and I have no plans to expand it beyond 2 people.
